## 3. 향후 참고 사항

### 3.1 Google Drive + Git 주의점
- 이 폴더는 Google Drive 동기화 영역이므로 `.git/` 폴더도 Drive에 동기화됨
- 동기화 중 충돌 파일(0바이트 `{GUID}...`)이 계속 생성될 수 있음 → `.gitignore`의 `{*}*` 패턴으로 처리 중
- 다른 PC에서 동일 Drive로 접근할 경우 `.git` 내부 충돌 가능성이 있으니 한 PC에서만 git 작업 권장

### 3.2 API 키 관리
- `ps-docparser/.env`에 실제 Gemini API 키 존재 — 현재 미공개 상태 확인됨
- 공개 저장소로 전환 시 `.env`가 절대 노출되지 않도록 재확인 필요
- `.env.example`은 `.gitignore`에서 예외 처리됨(추적 대상)

### 3.3 추가 작업 후보
- `README.md` 내용 보강 (프로젝트 개요, 설치·실행 방법, Phase 로드맵 링크)
