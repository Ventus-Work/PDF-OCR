## 5. Google Drive 동기화 주의점

### 5.1 충돌 파일 재발 가능성
- 이 폴더는 Google Drive 동기화 영역이므로 `.git/`, coverage, 로그 파일에도 충돌이 생길 수 있음
- 충돌 파일은 보통 `{GUID}...` 형식으로 생성되며 대부분 0바이트 또는 임시 조각 파일임
- 발견 시 우선 확인 후 정리

### 5.2 Git 작업 권장 방식
- 가능하면 한 PC에서만 git 작업
- 대량 테스트 직후에는 커밋 전 `git status`와 루트 파일 목록을 같이 확인
- Drive 동기화 직후에는 `{GUID}...` 파일이 생기지 않았는지 먼저 점검
