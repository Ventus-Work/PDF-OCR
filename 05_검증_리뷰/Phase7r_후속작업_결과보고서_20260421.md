## 3. 작업 C — `slow` marker 활성화

### 목적
MistralAI SDK 임포트 지연으로 인해 5개 테스트가 각 1~5초 소요. 기본 CI 러닝에서 제외하여 피드백 루프 단축.

### 구현 사항

**`tests/unit/engines/test_mistral_engine.py`**

```python
@pytest.mark.slow
class TestMistralEngineInit: ...

@pytest.mark.slow
class TestMistralEngineOcrDocument: ...

@pytest.mark.slow
class TestMistralEngineOcrImage: ...
```

### 동작 원리

`conftest.py` 의 `pytest_collection_modifyitems` 가 이미 `--run-slow` 옵션 없이 `slow` 마커 테스트를 자동 skip 처리하는 로직을 보유하고 있었음. 별도의 `pytest.ini` `addopts` 변경 없이 마커 추가만으로 충분.

```python
# tests/conftest.py (기존 코드)
def pytest_collection_modifyitems(config, items):
    run_slow = config.getoption("--run-slow", default=False)
    skip_slow = pytest.mark.skip(reason="--run-slow 옵션으로 실행")
    for item in items:
        if "slow" in item.keywords and not run_slow:
            item.add_marker(skip_slow)
```

| `pytest --run-slow` | 270 | 1 (기존 1) | ~24s |

## 4. 작업 D — `google.generativeai` → `google.genai` 마이그레이션

### 목적
`google.generativeai` 패키지 지원 종료로 발생하는 FutureWarning 해소. `google.genai 1.63.0` 이 이미 설치되어 있어 즉시 적용 가능.

### API 변경 대조

| 항목 | 구 API | 신 API |
|---|---|---|
| import | `import google.generativeai as genai` | `import google.genai as genai` |
| 초기화 | `genai.configure(api_key=key)` | `client = genai.Client(api_key=key)` |
| 모델 객체 | `genai.GenerativeModel(name)` | 없음 (client 에 통합) |
| 생성 호출 | `model.generate_content([...], safety_settings=LIST)` | `client.models.generate_content(model=..., contents=[...], config=Config(...))` |
| safety_settings | dict 리스트 | `types.SafetySetting(...)` 객체 리스트 |
| `response.text` | `response.text` | 동일 |
| usage_metadata | `prompt_token_count`, `candidates_token_count` | 동일 |

### 변경 파일

#### `engines/gemini_engine.py`

- `SAFETY_SETTINGS` (dict 리스트) → `SAFETY_SETTING_DEFS` (tuple 리스트) + `__init__` 에서 `types.SafetySetting` 객체로 변환
- `genai.configure()` + `self._genai` 제거 → `self._client = genai.Client(api_key=...)`
- `model = self._genai.GenerativeModel(...)` → 제거
- `model.generate_content([...], safety_settings=...)` → `self._client.models.generate_content(model=..., contents=[...], config=GenerateContentConfig(...))`

#### `tests/unit/engines/test_gemini_engine.py`

구 패턴:
```python
engine._genai = MagicMock()
mock_genai.GenerativeModel.return_value = mock_model
mock_model.generate_content.return_value = mock_response
```

신 패턴:
```python
engine._client = MagicMock()
mock_client.models.generate_content.return_value = mock_response
```

#### `requirements.txt`

```diff
- google-generativeai>=0.8.0,<1.0.0
+ google-genai>=1.0.0,<2.0.0
```
